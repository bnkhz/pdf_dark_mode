# Dark Mode PDF Converter

A small and lightweight tool that converts any PDF file into **dark mode**, improving readability and reducing eye strain.  
The converter processes PDFs by rendering pages as images, applying color inversion and enhancements, and rebuilding the PDF.

This project is written in **Julia**, which is required to run the tool.

## Features

- Convert standard PDFs to dark mode
- Improves reading comfort in low‑light environments
- Simple and scriptable workflow
- Works with most PDF files

## Requirements

The following programs **must be installed separately** and **downloaded from their official websites**:

- **Julia** (required to run the program)  
  https://julialang.org/

- **Poppler** (PDF rendering utilities)  
  https://poppler.freedesktop.org/

- **ImageMagick** (image processing)  
  https://imagemagick.org/

Make sure all tools are accessible from your system’s PATH.

## Installation

1. Install **Julia** from the official website.
2. Install **Poppler** and **ImageMagick** from their respective official sources.
3. Clone this repository.
